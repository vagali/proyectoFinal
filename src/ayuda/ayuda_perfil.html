Información básica

Esta ventana permite Modificar y actualizar los datos del usuario.

Requirimientos de actalizacion

Para modificar los datos de un usuario se podra acceder desde el boton Modificar perfil, en caso de que un campo sea introducido incorrectamente se le notificara el problema en cuestion.
Para la correcta actualizacion se debera introducir un Email correcto con su formato correcto ej: pedro.pedrin@gmail.com.
Nombre de usuario tendrá que ser único, en caso contrario se le notificará.
Contraseña tendra que tener mínimo 3 caracteres y se tendrá que repetir en confirmar contraseñas y tienen que coincidir. Una vez realizadas las acciones anteriores, deberá pulsar el botón Aceptar para actualizar datos del usuario. Si no se produce ningún error, se habra completado correctament el update.

Ingresar Saldo

Pulsando el botón Ingresar Saldo se abrira la ventana de ingreso de saldo.

Barra de tareas

Pulsando en algun componente del mismo se podra navegar entre las distintas ventanas.


Acceso rápido

Teclas Informacion
F1 Abre el menu de ayuda
Esc Salida rapida
Ctrl+F Abre buscador de archivos para subir imagenes.
Ctrl+P Hace visible campos de actualizar contraseña y al mismo tiempo los hace invisible.
Ctrl+U Hace visible campos de actualizar perfil y tambien los oculta.
Ctrl+I Abre la ventana de ingresar saldo.
